Formulario dinámico

            

                

                    

                        Escribir un programa que pida por prompt sucesivamente, los campos que se van a pedir en un formulario.
                        Cuando se ingrese "" (string vacío), se considera que ya no se requieren más campos.
                    

                    

                        (Ejemplo: "nombre", "apellido", "edad").
                    

                    

                        Luego, pedir por prompt esos datos y agregarlos a un objeto, y, cuando se hayan pedido todos, mostrar
                        el resultado.
                    

                    

                        Ejemplo:
                    

                    
                        {
                            nombre: 'Alberto',
                            apellido: 'Tapia',
                            edad: 48
                        }
